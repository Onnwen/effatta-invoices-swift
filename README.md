# EffattaInvoicesSwift

A lightweight Swift wrapper for the [Effatta Fatture](https://effatta.it) API.

## Features

- Send invoice to SDI
- Cancel an invoice

## Installation

Use [Swift Package Manager](https://swift.org/package-manager/) to add the dependency:

```swift
.package(url: "https://github.com/Onnwen/effatta-invoices-swift.git", from: "1.0.0")
```

Then add "EffattaInvoicesSwift" as a dependency for your target.
